# SoleStride - Shoe Store Website

## 📌 Project Overview

**SoleStride** is a complete e-commerce website for a shoe brand. It includes all essential pages for an online store with a clean, modern design.

## 🛠️ Technologies Used

- **HTML5** - Page structure
- **CSS3** - Styling (single stylesheet for all pages)
- **JavaScript** - Interactivity and dynamic content
- **Font Awesome 4.7** - Icons

## 📁 File Structure

```
SoleStride/
│
├── index.html                   # Homepage
├── shop.html                    # Shop/catalog page
├── PDP.html                     # Product details page
├── about.html                   # About us page
├── contact.html                 # Contact page
├── privacypolicy.html           # JavaScript file
├── Guidance(size guid).hmtl     # JavaScript file
├── ALL CSS files.css            # ALL CSS file separate for each page
├── data.js                      # JavaScript file
├── main.js                      # JavaScript file
└── README.md                    # This file
```

## 🚀 How to Run

1. Download all files to a folder
2. Open `index.html` in any web browser
3. Navigate using the menu bar

**No server or installation needed!**

## 🎨 Features

- ✅ Responsive design (works on mobile, tablet, desktop)
- ✅ Product image gallery
